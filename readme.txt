■プログラムの説明
・face.py：顔写真（男性画像：./men_pic,女性画像：./women_pic）から画像を読み込んで、
　　　　　CNNでディープラーニングしモデルを生成するプログラム。
　　　　　作成したモデルは「my_model.h5」として出力する。
・facejudge.py：17行目に指定した画像ファイルを作成したモデル「my_model.h5」を使って、
　　　　　　　　「女性らしさ」を判定するプログラム
　　　　　　　　
■プログラム実行前の準備　　　　　　　　
pythonプログラムと同じディレクトリに、「men_pic」「women_pic」というフォルダを作成して
そこに男性画像と女性画像を突っ込んでください。画像は縦横100pxのものにしてください。
